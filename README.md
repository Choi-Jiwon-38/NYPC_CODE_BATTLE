기댓값 테이블 계산

1. generator.py
- 모든 주사위의 경우의 수에 대한 기댓값을 계산함. 매핑 방식은 list[round][mask][upper_score]. (mask는 규칙 사용 여부를 나타내는 비트맵, upper_score는 기본 규칙 점수의 합)

2. convert.py
- 실제 제출에서는 data.bin으로 제출이 되므로, json 테이블 파일을 data.bin으로 변환함. 이때, 변환만 하면 용량이 제출하기에 너무 크므로 gzip 라이브러리로 압축하여 변환.

3. Yacht.py
- 기댓값 테이블을 계산에 사용하도록 변경함. 이때 테이블은 압축되어 있는 상태이므로, 압축을 해제하여 사용함.
